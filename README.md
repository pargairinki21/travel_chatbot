# Travel Agent Pro - Dashboard UI 

The interactive interface for the Travel Agent Pro system. Built with a focus on **Geospatial Visualization** and **Real-time AI Feedback**.

## UI Features
- **Split-Screen Dashboard**: Chat interface on the left, interactive Leaflet.js map on the right.
- **Magical Weather Modal**: A dynamic, glowing AI alert that blurs the background and updates its color (Green for Good, Red for Caution) based on real-time weather conditions.
- **Animated Route Tracer**: Visualizes the path from your origin to the destination with a moving dashed polyline.
- **Side Detail Drawer**: Glassmorphism panel that slides out to show high-resolution destination photos, travel tips, and "Must Try" local food.

## Tech Stack
- **Frontend**: HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript
- **Maps**: Leaflet.js
- **Animations**: CSS Keyframes & Cubic-Bezier transitions
- **Images**: LoremFlickr Dynamic Source

## Setup
1. Open `script.js`.
2. Update the `BACKEND` variable to point to your live Render URL.
3. Open `index.html` in any browser or use a Live Server.
